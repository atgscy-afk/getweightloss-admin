## System Basics

**What is this system?** The GetWeightLoss Admin Portal is the back-office for managing customer orders, profiles, communications, and sales operations. Only staff with an approved account can access it.

**What happens here?** 
- Sales agents find new leads and assign them to themselves or team members
- Support agents help customers with orders, documents, and inquiries
- Commission tracking shows sales agent performance
